# 五大壓力來源

來自 [[Stuart-Shanker|Dr. Stuart Shanker]]《孩子不是壞，只是壓力大》的核心框架。用於解釋孩子情緒崩潰的根本原因。適用年齡：0 歲 ～ 青春期。

## 五個類別

### 生理壓力
身體層面的不舒服：飢餓、疲勞、生病、感官過載。藥物副作用亦屬此類（如孟魯斯特 Montelukast 等過敏/氣喘藥，約 1–10% 的孩子服用後性格明顯改變；停藥即恢復）。

### 情緒壓力
環境中的情緒氛圍影響：父母的表情、語氣、家庭氣氛、他人的情緒。

### 認知壓力
理想與現實的落差：想做好卻做不到、被反覆要求卻達不到標準。

### 社交壓力
社交互動耗能：維持人際關係、遵守規則、應付群體，回家後能量耗盡而崩潰。

### 道德壓力
被賦予超出承受能力的道德責任：「你是老大，要讓弟妹」、代表老師照顧他人。

## 年齡與壓力的關係

年紀越小，生理壓力比重越高；隨年齡增長，更高層次的壓力類別逐漸出現：

```
1 個月 → 生理
1 歲   → + 情緒
2 歲   → + 認知
3 歲   → + 社交
5 歲+  → + 道德
```

## 使用方式

孩子崩潰時，逐一排查五個類別，找出當下最可能的主要壓力源，再做對應處理。
