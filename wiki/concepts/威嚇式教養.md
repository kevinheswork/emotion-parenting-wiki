# 威嚇式教養

## 定義

以恐嚇、大聲罵、惡狠眼神、拍桌子、人身攻擊等**心理威脅**手段讓孩子就範的管教方式。不直接使用肢體，但本質上仍是行為主義的「懲罰制約」。

在學校常見於：麥克風大聲喝斥、拍桌子、言語羞辱等方式。

---

## 五大副作用

### 1. 成效短暫

**玩具實驗**（育兒全百科引用）：
- 威嚇情境：校長拍桌大吼「不準碰玩具」
- 結果：6 週後，**77% 的孩子回到房間後通通違反規定**，去摸了那個玩具
- 說服情境：校長以道理說服孩子「這個玩具很珍貴要送給別人」
- 結果：6 週後，大部分孩子**仍記得並遵守**約定

結論：**說服式管教的長期成功率，是威嚇的兩倍以上。**

### 2. 惡性循環

威嚇需要不斷加重刺激強度才能維持效果：
- 一個眼神 → 拍桌子 → 大吼 → 罵人 → 人身攻擊 → 語言羞辱
- 父母在不知不覺中滑坡，自問「我怎麼能對孩子講出這麼惡毒的話？」
- 與[[體罰]]的惡性循環相同，有時會不自知地滑入兒童虐待範疇

### 3. 培養說謊與逃避

孩子面對威嚇的理性策略是「陽奉陰違」：
- 黃醫師自述：曾連續半學期模仿媽媽簽名（逃避被念的壓力，非成績問題）
- 孩子 18 年都在**練習逃避**，而不是練習探索內在動機
- 最終，孩子的成就感來源變成「騙過大人」，而非真正的自我成長

> 「這是很恐怖的事情。我們希望我們的孩子把時間花在探索自己的生命，然後打造一個內在動機，而不是打造說謊的功夫或表面功夫。」

### 4. 反社會傾向

威嚇讓孩子對「被威嚇的事物」產生負面連結，並不斷擴散：

**喝湯實驗**（育兒全百科引用）：
- 被威嚇喝湯的孩子，**討厭那碗湯的比例增加兩倍**（連本來不挑食的也開始討厭）

擴散路徑：
> 討厭考試 → 討厭學科（英文）→ 討厭學校 → 討厭老師同學 → 討厭政府 → 討厭社會

黃醫師認為，台灣社會「仇師氣氛最濃厚的時代」正是上一代打罵教育的遺毒。

### 5. 暴力即權力的示範

威嚇式教養向孩子傳遞一個核心信念：**「暴力/權威可以讓人就範，這是有效的生存技能」**。

- 孩子會在手足間複製：老大學父母的方式「威嚇」弟妹
- 表面看起來可愛，實際上是孩子學到的**第一個領導風格**
- 長大後可能在社會上也傾向以暴力或脅迫爭取利益

## 與體罰的比較

| | 體罰 | 威嚇式教養 |
|---|------|------------|
| 手段 | 肢體疼痛 | 心理恐嚇、語言羞辱 |
| 短期效果 | 有（暫時就範）| 有（暫時就範）|
| 長期效果 | 無效 / 反效果 | 無效 / 反效果 |
| 惡性循環 | ✓（需要越打越重）| ✓（需要越罵越狠）|
| 兒虐風險 | 高 | 中（語言虐待）|

兩者都是行為主義的「懲罰」機制，只是媒介不同，副作用結構高度相似。
